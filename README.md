# 스마트 암기카드 - Frontend
 
## 기술 스택
`React` `JavaScript`
 
## 주요 기능
- 그룹/카드 관리 화면 구현 (생성, 삭제)
- 학습 모드(일반/스마트), 진행 방식(순차/랜덤) 선택 기능
- 카드 플립, 별표 토글 등 인터랙션 및 모바일 환경 지원
  
## 인프라
- AWS EC2 서버 배포
